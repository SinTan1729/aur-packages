## AUR Packages

A backup of my [AUR packages](https://aur.archlinux.org/packages?K=Sintan&SeB=m).

To push to AUR, first do `makepkg --srcinfo > .SRCINFO `, then add and commit the
`PKGBUILD` and `.SRCINFO` files, finally push them to AUR.
